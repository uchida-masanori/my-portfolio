帰国を助けたTodoManager
Git
AngularとFirebaseで作成したタスク管理アプリです。タスクを作成して、todo / doing / doneの３種類のステータスに分けて管理することができます。タスクはドラッグ＆ドロップでステータスを変更できます。Googleアカウントを利用したログイン機能もつけてみました。
今回、実際に日本帰国に必要な作業の管理に使いました。非常にシンプルな仕組みですが、帰国準備程度であれば十分な機能かと思います。
改善点としては、タスクの重要度を色で振り分ける機能とタスクの期限設定/通知機能があるとさらに実用的になるかと感じました。せっかくですから、改良して今後も旅の友として使っていきたいです。
懐かしのポケモン図鑑
Git
ポケモンをクリックすると対象のポケモンのタイプと個体値が表示されるポケモン図鑑を作成しました。AngularMaterialを利用したSPAの作成とJson APIを使った基礎学習を狙いました。今回はレスポンシブにも対応しております。
Jsonファイルからポケモンの個体値を引いてくる作業の中で、型のエラーが思ったよりも多く発生しました。キャストやパイプについて調査時間を取られましたが、ググり方と使い方を学べた良い作品となったと思います。
ポケモン一覧のデザインや個体値をチャートにまとめる作業も簡単にできて、AngularMaterialの便利さを改めて感じました。
Firebase無料版で使用できる容量を超えてしまったので、ポケモン図鑑はデプロイせずGitのみの公開です。
